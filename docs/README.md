# Documentazione

Indice rapido dei file di supporto:

- `HANDOFF_CLAUDE_CODE.md`: handoff rapido e operativo per un agente esterno che entra nel progetto.
- `RIEPILOGO_PROGETTO.md`: panoramica completa del progetto, dell'architettura e dei flussi.
- `STRUTTURA_PROGETTO.md`: spiega come e perche sono state separate cartelle, codice e dati.
- `FLUSSO_OSM.md`: descrive il flusso automatico basato su OpenStreetMap e Overpass.
- `FLUSSO_LINKEDIN.md`: descrive il flusso manuale assistito da LinkedIn.
- `FLUSSO_RICERCA_ONLINE.md`: descrive il flusso che arricchisce il CSV cercando informazioni sul web.
- `FLUSSO_MESSAGGI_MIRATI.md`: descrive il flusso per analisi commerciale e messaggi personalizzati.
- `FLUSSO_CHATGPT_DEEP_RESEARCH.md`: descrive come preparare un batch ridotto da dare a ChatGPT / Deep Research.
- `PROMPT_CHATGPT_DEEP_RESEARCH.md`: prompt gia pronto da incollare in ChatGPT / Deep Research.
- `STRATEGIA_ACQUISIZIONE.md`: note commerciali e operative.

Per i file dati consulta anche `../data/README.md`.
