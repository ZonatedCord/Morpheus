# Documentazione

Indice rapido dei file di supporto:

- `HANDOFF_CLAUDE_CODE.md`: handoff operativo per un altro modello o agente.
- `RIEPILOGO_PROGETTO.md`: panoramica completa dell'architettura attuale, inclusi SQLite multi-dataset e frontend React.
- `STRUTTURA_PROGETTO.md`: struttura del repo e ruolo di backend, frontend e pipeline CSV.
- `FLUSSO_OSM.md`: flusso automatico basato su OpenStreetMap e Overpass.
- `FLUSSO_LINKEDIN.md`: flusso manuale assistito da LinkedIn.
- `FLUSSO_RICERCA_ONLINE.md`: arricchimento online best-effort.
- `FLUSSO_MESSAGGI_MIRATI.md`: analisi commerciale e messaggi personalizzati.
- `FLUSSO_CHATGPT_DEEP_RESEARCH.md`: preparazione batch per ChatGPT / Deep Research.
- `PROMPT_CHATGPT_DEEP_RESEARCH.md`: prompt pronto per ChatGPT / Deep Research.
- `STRATEGIA_ACQUISIZIONE.md`: note commerciali e operative.

Per i file dati consulta anche `../data/README.md`.
